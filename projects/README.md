## 通用教训（跨项目）

### 截图
- 先发文字激活通道，再发图片，成功率高
- 字体加载超时会阻塞截图
- 知识卡片用固定viewport，信息图用全页截图

### exec工具
- code !== 0 要明确告知用户失败

### 内容真实性
- 严禁编造，所有数据需核实来源
- 宁可留空，不能写错

### 微信发图
- 先发文字激活通道，再发图片
