## 二、信息图规范

**文件：** `memory/infographic-strategy.md`

**触发词：** 信息图、新闻图、热点图、数据图、资讯图

**核心特点：**
- 多模块拼贴（6-8个模块）
- 模块化布局：标题 + 内容块 + 统计数据
- 严禁留白，内容填满
- 第8模块固定为 Jarvis 分析评估（紫色[AI]主题）

**布局规则：**
- 等宽：6列网格，3列×2列交替
- 创意不等宽：h6/h4/h3/h2 混排（示例：6→3+3→3+3→4+2→3+3→6）
- accent bar：3px左侧渐变
- rank：右上角大号半透明

**字号规范：**
| 元素 | 字号 |
|------|------|
| 年份数字 | 88px |
| 主标题 | 48px |
| Hero标题 | 40-45px |
| 正文 | 19px |
| 统计数字 | 20-22px |

**配色：**
```css
--black:  #0a0a0a;
--white:  #fafafa;
--accent: #ff3366;  /* 可随主题变化 */
```

**字体：** Inter（英文）+ Noto Sans SC（中文）

**截图：** viewport 900×1200（竖屏3:4），device_scale_factor=2

## 三、知识卡片规范

**文件：** `skills/knowledge-card/layout-strategy.md`

**触发词：** 知识卡片、深度卡片、思考卡片、科普卡片

**核心特点：**
- 三段式结构（32% / 52% / 16%）
- 顶部：深黑底 + 标题渐变
- 中部：白底三栏（01/02/03）
- 底部：金色引言条

**布局规则：**
- 顶部32%：kick标签 + 大标题(75px) + 副标题 + 右上装饰年份
- 中部52%：三栏等分，数字锚点 + 标题 + 正文 + 标签
- 底部16%：金色背景 + 引言金句 + 出处

**字号规范：**
| 元素 | 字号 |
|------|------|
| 主标题 | 75px |
| 装饰年份 | 88px |
| 栏目标题 | 18px |
| 正文 | 17px |
| kicker/标签 | 9px |

**配色：**
```css
--black:  #141414;       /* 顶部背景 */
--white:  #FAFAF8;       /* 中部背景 */
--accent: #E8C547;       /* 底部/数字锚点 */
--gray-light: #E2DFD6;   /* 分割线 */
--gray-mid:  #C4BFA0;    /* 栏目标签 */
--gray-dark: #888888;    /* 正文 */
```

**字体：** Noto Serif SC（标题）+ Noto Sans SC（正文）

**截图：** viewport 800×1067（精确匹配），device_scale_factor=2，自动裁剪顶部/底部各2px白边，输出 1600×2130px

## 四、选择指南

**选信息图当：**
- 内容是热点资讯、数据聚合
- 需要展示多个独立信息点
- 强调可看性和传播性
- 示例：NBA赛事综述、Reddit热门话题、科技新闻周报

**选知识卡片当：**
- 内容是知识提炼、深度洞察
- 围绕一个主题展开三维度分析
- 强调思考性和启发性
- 示例：为什么学习击剑、认知心理学、亲子关系方法论
