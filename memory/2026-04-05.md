### 最终确定的规范

**字体**
- 主字体：`Inter`（英文/数字，UI之王）
- 中文：自动 fallback 到 NotoSansSC
- 本地路径：`/home/node/.fonts/custom/Inter-Regular.ttf` + `Inter-Bold.ttf`

**字号**
- 正文（card-body）：19px
- 数字统计（stat）：20px
- Hero 统计（hero.stat）：22px
- 年份数字：88px
- 主标题：48px
- Hero 标题：40-45px

**配色系统**
```css
--black: #0a0a0a
--white: #fafafa
--accent: #ff3366  /* 可随主题变化 */
```

**布局**
- 3:4 竖屏，6列网格
- 严禁留白！内容填满整个页面
- **8个模块**（最终版，用户确认最满意）
- 等宽布局：6列网格，每行3列×2列交替
- 创意不等宽：h6/h4/h3/h2 混排（示例：6→3+3→4+2→3+3）
- **第8模块固定为 Jarvis 分析评估**（紫色 [AI] 主题，宏观/外交/内政/亮点 + 高/中/低评分）

**卡片结构**
```
[accent bar] | [rank] [icon][title][sub]
             | [card body]
             | [stats]
             | [tags]
```
- accent bar：3px，左侧，glow 阴影
- rank：右上角，大号半透明 opacity 0.08
- icon：30px 圆角，符号代替 emoji（不用emoji）
- tags：胶囊标签

**截图规范**
- 3:4 竖屏：视口 900×1200，max-width 900px
- 4:3 横屏：视口 1200×900，max-width 1200px
- 视口必须等于内容尺寸，否则留白

**发送技巧**
- 先发文字消息激活通道，再发图片，成功率更高

**背景装饰**
```css
background:
  radial-gradient(ellipse at 10% 90%, rgba(accent, 0.20) 0%, transparent 50%),
  radial-gradient(ellipse at 90% 10%, rgba(secondary, 0.15) 0%, transparent 50%);
```
