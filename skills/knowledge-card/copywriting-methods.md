# 知识卡片文案方法论

> 调研整理，2026-04-06

---

## 一、底层逻辑

### 必要难度理论
- 存入知识时越"吃力"，调用时越牢固
- 知识卡片 = 提高存入难度 = 过脑不过眼
- 结构化 + 视觉化 = 加深记忆

### 卢曼卡片笔记法核心
- **原子化**：每张卡片只装一个完整概念
- **自下而上**：不预设分类，让知识自然生长
- **链接优先**：卡片之间建立网状关联

## 二、文案编辑方法

### 标题公式

**公式：** `[主题] + [独特视角/价值承诺]`

**好标题标准：**
- 具体不抽象
- 有价值承诺
- 引发好奇或认同
- 简单准确，不做"标题党"

### 三栏内容公式

每栏 = `数字锚点` + `短标题` + `关键词标签` + `一句话正文`

**结构规范：**
- 数字锚点：01 / 02 / 03，视觉节奏感
- 短标题：≤12字，一句话概括一个维度
- 关键词：3-5个，体现内容维度
- 正文：≤30字，一句话讲清楚

### 金句公式

**引言 = 一句话洞察 + 来源署名**

好的引言：
- 有洞察力，能引发共鸣
- 不冗长，一句话讲完
- 有立场，不说废话

## 三、内容提炼技巧

### NanoBanana2 三步法

1. **备内容** — 把复杂内容拆成独立知识点，决定几张卡片
2. **定风格** — 选风格 + 补充详细提示词
3. **第一张定风格** — 确认后反推提示词，保持系列统一

### 自媒体内容技巧

**开头：引人入胜**
- 一个有趣的例子
- 一个反常识的洞察
- 一个痛点共鸣

**结构：线性叙事 or 非线性叙事**
- 线性：按时间/因果/逻辑顺序
- 非线性：碎片化，并列排布

**转化：给读者一个行动理由**
- 知识类 → 收藏转发
- 情感类 → 点赞评论
- 工具类 → 保存备用

## 四、AI提示词模板

### 通用提示词框架

```
你是一位资深内容策划专家，擅长将复杂信息提炼为简洁有力的知识卡片。

请根据以下内容素材，生成一张知识杂志卡片的内容：

【原文/主题】：{内容}

【目标读者】：
【卡片用途】：朋友圈 / 小红书 / 公众号 / 微博

请生成：
1. 主标题（≤15字，有冲击力，具体明确）
2. 三栏内容（每栏：短标题≤12字 + 关键词3-5个 + 正文≤30字）
3. 金句引言（一句话洞察，有共鸣或反常识）

要求：
- 语言精炼，不说废话
- 角度独特，不是总结而是提炼观点
- 有信息增量，读者能学到东西
```

```

主题：
数量：N张

每张内容：
- 主标题：
- 三个知识点（短标题 + 关键词 + 正文）：
- 金句引言：

风格：杂志风（深黑顶 + 白底三栏 + 金色引言条）
```

## 五、知识卡片分类体系

来自少数派/卢曼笔记法：

| 卡片类型 | 说明 |
|---------|------|
| 概念卡 | 定义一个核心概念 |
| 观点卡 | 一个独特视角或立场 |
| 案例卡 | 具体实例/故事 |
| 方法卡 | 操作步骤/流程 |
| 金句卡 | 精华引言/摘录 |

## 六、自媒体适配建议

### 平台特性
| 平台 | 风格偏好 |
|------|---------|
| 小红书 | 实用感强、视觉冲击、轻松亲切 |
| 公众号 | 深度有料、专业感、阅读节奏 |
| 朋友圈 | 简洁有力、一眼看懂、收藏导向 |
| 微博 | 短平快、有话题性、互动感 |

### 内容角度选择
- **WHY（为什么）**：改变认知，引发共鸣
- **HOW（怎么做）**：实用工具，拿来即用
- **WHAT（是什么）**：概念澄清，有趣有料
