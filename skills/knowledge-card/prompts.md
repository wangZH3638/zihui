## 风格提示词

### 1. 卡通扁平化

```
视觉风格：卡通扁平化 + 清晰模块分区
比例：竖版 3:4
配色方案：
  主背景：纯白色 #FFFFFF
  模块色块：
    模块一：浅蓝底色 #E6F7FF + 深蓝边框 #1890FF
    模块二：浅紫底色 #F9F0FF + 深紫边框 #722ED1
    模块三：浅绿底色 #F0FFF4 + 深绿边框 #52C41A
文字风格：
  标题：深灰 #333333，加粗醒目
  正文：中灰 #666666，清晰易读
  核心数据：橙色 #FA8C16，突出重点
视觉元素：
  每个模块角落添加卡通小装饰
  模块间用简约虚线分隔
  每类内容搭配对应属性小图标
```

### 2. 软UI新拟态风

```
视觉风格：软UI新拟态风
比例：竖版 3:4
配色方案：低饱和度莫兰迪色系
模块色块：
  模块一：浅蓝底色 #E6F7FF + 深蓝边框 #1890FF
  模块二：浅紫底色 #F9F0FF + 深紫边框 #722ED1
文字风格：
  无衬线字体：思源黑体 /苹方
  层级分明：标题大+深色，正文适中，核心数据橙色
视觉元素：新拟态圆角卡片，线性图标，微质感背景，柔和分隔线
```

### 3. 商务深色风（现有信息图风格）

```
视觉风格：深色商务风
比例：竖版 3:4
配色方案：
  主背景：深色 #0D0D14
  模块色块：
    模块一：浅蓝底色 rgba(24,144,255,0.15) + 深蓝边框 #1890FF
    模块二：浅紫底色 rgba(114,46,209,0.15) + 深紫边框 #722ED1
  强调条：3px 左侧渐变
文字风格：
  标题：白 #FFFFFF，加粗
  正文：半透明白 rgba(255,255,255,0.5)
  核心数据：亮色 #1890FF / #FA8C16
视觉元素：左侧彩色强调条、rank编号、右上角大号半透明数字
```

### 4. 国潮风

```
视觉风格：国潮风 + 现代排版
比例：竖版 3:4
配色方案：
  主背景：朱红 #C53D43 / 墨色 #1A1A2E
  点缀：金色 #D4AF37 / 青色 #00A69C
  文字：米白 #F5E6D3
视觉元素：传统云纹、祥云、山水剪影、印章
字体：方正趣宋体 / 思源宋体
```

### 5. 毛玻璃风

```
视觉风格：Glassmorphism 毛玻璃风
比例：竖版 3:4
配色方案：
  主背景：渐变深色 #0D0D1A → #1A1A3E
  毛玻璃卡片：rgba(255,255,255,0.08) + backdrop-blur(20px)
  边框：rgba(255,255,255,0.15)
  强调：渐变青紫 #00D4FF → #7B61FF
视觉元素：圆角大卡片、透明玻璃质感背景、光晕
```

### 6. 复古未来主义

```
视觉风格：复古未来主义 Y2K
比例：竖版 3:4

  主背景：纯黑 #000000 + 霓虹渐变
  强调：金属银 #C0C0C0 + 电光粉 #FF00FF + 青蓝 #00FFFF
文字风格：
  标题：金属质感渐变，大字号
```
