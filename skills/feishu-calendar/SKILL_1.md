# 飞书日历助手

快速管理飞书日历。

## 触发条件

用户说：
- 添加日程
- 创建日程
- 添加提醒
- 提醒我
- 查日历
- 今天有啥安排
- 明天有啥
- 这周有啥

## 功能

### 1. 添加日程

告诉我：
- 📝 标题（必填）
- 📅 日期和时间
- ⏰ 提醒时间（如：提前10分钟）
- 📍 地点（可选）
- 👥 参与者（可选）

**示例：**
- "明天上午9点开会"
- "今天下午3点提醒我做报告"
- "下周一上午10点 Review 会议"

### 2. 查看日程

**示例：**
- "今天有啥"
- "明天有啥安排"
- "这周会议"
- "查看日历"

### 3. 删除/修改日程

**示例：**
- "删除明天的会议"
- "取消下午3点的日程"

## 日程格式

飞书支持多种格式：

| 类型 | 示例 |
|------|------|
| 单次 | 2026-03-20 14:00 |
| 重复 | 每天 / 每周一 / 每月15号 |

## 提醒设置

| 提醒时间 | 说明 |
|----------|------|
| 0 | 不提醒 |
| 5 | 提前5分钟 |
| 10 | 提前10分钟 |
| 15 | 提前15分钟 |
| 30 | 提前30分钟 |
| 60 | 提前1小时 |

## 使用流程

1. 用户提出需求
2. 提取关键信息：标题、时间、提醒
3. 调用 feishu calendar API 创建
4. 返回结果确认
